## Weekly Workflow

Each week folder is intended to hold:
- `notes/` for lesson notes and concepts
- `practice/` for labs, coding drills, and experiments
- `projects/` for show/project deliverables
- `reflection.md` for what I learned and how AI tools were used

Suggested routine:
1. Add lesson notes after class.
2. Store all practice code inside that week's folder.
3. Document debugging/refactoring decisions.
4. End with a short retrospective.

## Course Focus Areas

- AI-native programming workflows
- Debugging and refactoring AI-generated code
- Responsible use of AI coding tools
- Professional collaboration with Git and GitHub
- Core ML/LLM engineering concepts

## How This Repo Will Be Used

- Capture progress week-by-week

- Track growth in technical decision-making
- Keep evidence of iterative improvement over time
